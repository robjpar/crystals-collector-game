Crystals Collector Game

Instructions
A random number is generated at the start of the game. There are four crystals displayed as buttons on the page. Each has a specific number of points assigned to. When the player clicks on a crystal, the assigned number of points is added to the player's total score. The player wins if the total score matches the random number or loses if the total score goes above the random number. The game restarts whenever the player wins or loses.
